Описание решения:

  • • Источник данных: Media данные и Sales данные.
  • • Способ интеграции: Через External Tables в выделенную базу данных (аналитическое хранилище)
  • • Периодичность обновления: 1 раз в час.
  • • Логика обработки: Реализована средствами SQL внутри аналитического хранилища.
  • • Система визуализации: Yandex DataLens, подключенный к хранилищу.
  • • Инфраструктура: Аналитическое хранилище размещено на отдельном сервере (виртуальной машине) с сетевым доступом для передачи данных в Yandex Cloud.
